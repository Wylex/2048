# 2048

2048 game-like where:
> 2 -> 0
 4 -> 1
 8 -> 2
 16 -> 3
 32 -> 4
 64 -> 5
 128 -> 6
 256 -> 7
 512 -> 8
 1024 -> 9
 2048 -> 10

The merge of two equal squares means the number will be augmented by one.

Instructions (moving)
------------
Direction:  key<br />
> up:      W<br />
> left:    A<br />
> down:    S<br />
> right:   D<br />
